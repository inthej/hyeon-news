# 현뉴스 Backend

## Package
```
$ yarn add koa
$ yarn add koa-router
$ yarn add koa-cros
$ yarn add axios
$ yarn add dotenv
$ yarn add --dev nodemon
$ yarn add --dev @babel/cli @babel/core
$ yarn add koa-static
$ yarn add koa-send
```
