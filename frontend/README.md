# 현뉴스 Frontend

## Package
```
$ yarn create react-app frontend
$ yarn add react-router-dom
$ yarn add styled-components
$ yarn add axios
$ yarn add --dev --exact prettier
$ yarn add moment
$ yarn add prop-types
$ yarn add react-modal
```
